**🚍 Transport Reservation System (Java GUI)**

A desktop-based Transport Reservation System developed using Java Swing and AWT for the Object-Oriented Programming (OOP) course.<br>
The system provides a simple graphical interface for booking and managing transport tickets.

📌 Features<br>
🧾 User-friendly GUI built with Java Swing<br>
🚌 Make transport reservations<br>
🎫 Book tickets for available routes<br>
👤 Passenger information handling<br>
📅 Date and time selection for travel<br><br>

🛠️ Technologies Used<br>
Java (JDK 8+)<br>
Swing (javax.swing)<br>
AWT (java.awt)<br>
OOP Principles<br><br>
