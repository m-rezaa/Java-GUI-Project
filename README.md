**🚍 Transport Reservation System (Java GUI)**

A desktop-based Transport Reservation System developed using Java Swing and AWT for the Object-Oriented Programming (OOP) course (2nd Semester).
The system provides a simple graphical interface for booking and managing transport tickets.

📌 Features
🧾 User-friendly GUI built with Java Swing
🚌 Make transport reservations
🎫 Book tickets for available routes
👤 Passenger information handling
📅 Date and time selection for travel

🛠️ Technologies Used
Java (JDK 8+)
Swing (javax.swing)
AWT (java.awt)
OOP Principles
